# Cozy City Co. 🏙️

A visually stunning, minimalist 2D isometric city building game built with React and HTML5 Canvas.

![Cozy City Co. Preview](https://via.placeholder.com/800x450.png?text=Cozy+City+Co.+Preview)

## Features
- **Isometric Perspective**: Classic city-builder feel with a modern twist.
- **Creative Sandbox**: Build freely with roads, buildings, parks, and water features.
- **Geometric Aesthetics**: Minimalist, clean design using geometric shapes.
- **Smooth Interaction**: Pan, zoom, and place with intuitive mouse controls.
- **Connected Infrastructure**: Roads and water flow together seamlessly.

## Tech Stack
- **React**: UI Layer and state management.
- **Vite**: Ultra-fast development environment.
- **HTML5 Canvas**: High-performance game rendering.
- **Lucide React**: Beautiful minimalist icons.
- **CSS3**: Premium glassmorphism and modern styling.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/JoeMighty/Cozy-City-Co.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Controls
- **Left Click**: Place selected tool.
- **Middle Click / Shift + Left Click**: Pan camera.
- **Mouse Wheel**: Zoom in/out.
- **Tools**: Toggle between Roads, Buildings, Parks, and Water.
